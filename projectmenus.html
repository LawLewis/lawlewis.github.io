Advanced Menus

By Steven Lewis

What is it?

A project focused on creating different types of menus and going into detail on the creation to make them advanced and complex. Menus like a normal main menu, a 3D main menu, pause menus, level select menus and so on.

Features

Settings Menu - A menu that allows advanced changing of different settings like audio types, video settings, resolutions, graphics, controls.
